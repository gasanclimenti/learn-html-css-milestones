The a tag is a child of the p tag in the snippet.

The p tag signifies that a paragraph is beginning and opening. The /p tag closes the paragraph to mark what goes inside the paragraph, and for the code to be read and excecuted correctly.


Attributes on the a tag

  • href
  • id
  • class
  • target

Href attribute:

  • value = "https://google.com"
  • The href tag provides the specific location that the anchor tag will link to, in terms of a URL. In the snippet, the hyperlink will link to Google

ID attribute:

  • value = "the-best-link"
  • The id attribute labels a unique id, which is titled "the-best-link" in this situation. The id can be used in CSS to specifically alter a certain element.

Class attribute:

  • value = "banana"
  • The class attribute also provides a specific label for the element, though unlike the id, can be used multiple times with multiple other classes. It is also less specific than an id when used in CSS.

Target Attribute:

  • value = "_blank"
  • The target attribute determines the location of where a hyperlink will open (in this case, a seperate new tab) or where a form will submit to.
